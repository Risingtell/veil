# Veil — Demo Video: simple step-by-step

You will record ONE screen with ONE command. Aim for about 2–3 minutes.
Just follow the steps in order. The lines after 🎤 are what you SAY out loud.
The lines after ⌨️ are what you TYPE.

---

## PART 1 — Get ready (before recording)

1. Open the Windows Search bar, type **Git Bash**, press Enter. A black window opens.

2. ⌨️ Type this and press Enter:
   ```
   cd /c/Users/HP/veil
   ```

3. ⌨️ Type this and press Enter (this is a practice run — not recording yet):
   ```
   bash scripts/show.sh
   ```

4. Watch it run for about 8 seconds. It should end with the name
   **"Fatima Sani — Payroll #003"**. Run it 1–2 more times until you feel ready.

5. ⌨️ Type `clear` and press Enter to get a clean screen.

6. Make the window big (drag a corner) so the text is easy to read on video.

---

## PART 2 — Record

7. Press **Win + G**. The Game Bar appears. Click the round **Record** button.
   (If it asks permission the first time, click Yes.)

8. 🎤 SAY this (your intro):
   > "This is Veil — a private payments pool built on Stellar.
   > Normally, private payments look like money laundering, so banks avoid them.
   > Veil is different: every withdrawal also proves, using zero-knowledge,
   > that the money came from an approved, compliance-checked group.
   > Everything I'm about to show is live on the Stellar test network."

9. ⌨️ Type this and press Enter:
   ```
   bash scripts/show.sh
   ```

10. Let it finish (about 8 seconds). Then 🎤 SAY this while you slowly scroll up
    through the results:

    > "First — the proof is checked on-chain. The contract itself verified the
    > zero-knowledge proof and returned TRUE, using Stellar's new Protocol 26."

    > contract rejects it. No double-spends."

    > public, the records are just scrambled text. But the auditor can open them

11. 🎤 SAY this (your closing):
    > "Real zero-knowledge, verified on-chain, a real private payout, compliance
    > built in, and selective disclosure for regulators. That's Veil — exactly the
    > real-world zero-knowledge Stellar is asking for."

12. Press **Win + G** again, then click **Stop**.
